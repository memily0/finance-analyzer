# finance-analyzer
Backend-сервис на Kotlin/Spring Boot для анализа пользовательских транзакций. Реализованы загрузка и хранение транзакций, категоризация расходов, расчет monthly spending analytics, поиск аномальных трат и регулярных списаний, REST API для финансовых отчетов и пользовательских инсайтов.
